    Flappy BirdS is an experimental online multiplayer Flappy Bird game using node.js and Socket.IO. I took Nebez Briefkani's JS/CSS/HTML recreation of Dong Nguyen's original Flappy Bird game and added a backend component to it.

    This is the first iteration for Flappy BirdS. There is a lot of logic on the frontend that I would like to move to the backend. This has also introduced a lot of limitations to the multiplayer aspect of the game that you will notice as you play. Moving forwards, I plan to create my own clientside for Flappy BirdS and move most of the game logic to the backend.
